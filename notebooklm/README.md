# NotebookLM Integration

Ez a mappa NotebookLM-hez optimalizált dokumentumokat tartalmaz a projekt fejlesztéséhez és automatizálásához.

---

## Fájlok

### 1. VIDEO_SCRIPTS.md
Részletes videó scriptek a 4 demo videóhoz:
- Meeting Assistant - Gyors bemutatás (3-5 perc)
- Meeting Assistant - Részletes beállítás (10-15 perc)
- PMO Report Generator - Gyors bemutatás (3-5 perc)
- PMO Report Generator - Részletes beállítás (10-15 perc)

**Használat NotebookLM-ben:**
1. Töltsd be a fájlt NotebookLM-be
2. Add meg a demo környezet URL-jét
3. NotebookLM segítségével készítsd el a videókat a scriptek alapján

### 2. SCREENSHOT_GUIDE.md
Részletes útmutató 13 screenshot készítéséhez:
- Meeting Assistant: 7 képernyőkép
- PMO Report Generator: 6 képernyőkép

**Használat NotebookLM-ben:**
1. Töltsd be a fájlt NotebookLM-be
2. Add meg a demo környezet URL-jét
3. NotebookLM automatikusan készítheti a screenshot-okat

### 3. INTEGRATION_GUIDE.md
Teljes fejlesztési útmutató:
- Projekt setup
- OpenRouter integráció
- Meeting Assistant agent implementáció
- PMO Report Generator agent implementáció
- API endpoints
- Testing

**Használat NotebookLM-ben:**
1. Töltsd be a fájlt NotebookLM-be
2. Add meg a projekt könyvtárat
3. NotebookLM segítségével generáld le a kódot

---

## NotebookLM Használati útmutató

### 1. Videók készítése

```
1. Nyisd meg NotebookLM-et
2. Hozz létre egy új notebook-ot
3. Töltsd be a VIDEO_SCRIPTS.md fájlt
4. Add hozzá a demo környezet URL-jét
5. Kérj segítséget NotebookLM-től:
   - "Készítsd el a Meeting Assistant gyors bemutatás videóját"
   - "Generáld le a videó scriptet részletes beállításhoz"
```

### 2. Screenshots készítése

```
1. Töltsd be a SCREENSHOT_GUIDE.md fájlt NotebookLM-be
2. Add meg a demo környezet URL-jét
3. Kérj segítséget:
   - "Készítsd el az összes screenshot-ot a Meeting Assistant-hoz"
   - "Generáld le a PMO Report Generator screenshot-okat"
```

### 3. Kód generálás

```
1. Töltsd be az INTEGRATION_GUIDE.md fájlt NotebookLM-be
2. Add meg a projekt könyvtárat
3. Kérj segítséget:
   - "Hozd létre a Meeting Assistant agent.py fájlt"
   - "Generáld le a PMO Report Generator kódot"
   - "Készítsd el az API endpoint-okat"
```

## Best Practices

1. **Dokumentumok betöltése**: Mindig töltsd be a teljes dokumentumot, ne csak részleteket
2. **Kontextus megadása**: Add meg a projekt struktúráját és környezetét
3. **Iteratív fejlesztés**: Kérj visszajelzést NotebookLM-től és finomhangold
4. **Tesztelés**: Mindig teszteld a generált kódot/videót/screenshot-ot
