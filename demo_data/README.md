# Demo Data

Ez a mappa tartalmazza a demo környezethez szükséges teszt adatokat.

## Fájlok

### Meeting Assistant
- `meeting_demo_transcript.txt` - Demo meeting átirat
- `meeting_request.json` - API kérés példa meeting feldolgozáshoz
- `meeting_minutes_demo.json` - Várható AI output példa

### PMO Report Generator
- `jira_demo_data.json` - Demo Jira projekt adatok
- `budget_demo.xlsx` - Demo költségvetés Excel fájl
- `report_request.json` - API kérés példa riport generáláshoz
- `pmo_report_demo.json` - Várható AI output példa

### Adatbázis
- `init.sql` - Adatbázis inicializáló script
