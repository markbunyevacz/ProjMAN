## Megjegyzés

**JELENLEG PLACEHOLDER FÁJLOK**: Ezek a fájlok még nem tartalmaznak valós képernyőképeket. 
A tényleges képernyőképeket a platform implementálása után kell készíteni.

## Képernyőképek készítése

1. Futtasd a demo környezetet (`docker-compose.demo.yml`)
2. Kövesd a beállítási útmutató lépéseit
3. Készíts képernyőképeket minden lépésnél
4. Nevezd át a fájlokat a fenti elnevezési konvenció szerint
